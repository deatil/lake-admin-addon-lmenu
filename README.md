### 菜单结构插件


## 项目介绍

*  基于lake-admin的菜单结构插件，提取后台菜单分级结构格式


### 开源协议

*  `lake-admin` 遵循 `Apache2` 开源协议发布，在保留本系统版权（包括版权文件及系统相关标识，相关的标识需在明显位置标示出来）的情况下提供个人及商业免费使用。

## 使用方法 

*  正确安装了lake-admin后台
*  composer require lake/lake-admin-addon-lmenu，然后选择最新版本
*  进入后台安装该模块
